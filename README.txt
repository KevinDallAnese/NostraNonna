° Landing Page: Nostra Nonna °

El teatro "Nostra Nonna" es un templo ocultista ficticio, el cual brinda un espacio para la incorporación de habilidades histriónicas, espirituales y de razonamiento.
Tambien brinda espectáculos referentes a la tematica propia del establecimiento.
En esta landingpage tenemos un pantallazo donde se pueden ver estos elementos al igual que se hayan formas de comunicarse con la institución.

## Características del Proyecto

*Esta página es responsive, adaptandose de esta manera tanto a formato pc como otros dispositivos móviles.
*Cuenta con un loader, el cual muestra una pantalla de carga negra con un ojo rojo el cual realiza un Zoom In y que baja su opacidad al estar cargado el sitio.
*Al inicio de la página podemos ver una Navbar funcional, la cual cuenta con barra de busqueda y enlaces a distintas partes de la Landingpage.
*Tambien contamos con el uso de carruseles propios de Bootstrap. El primero lo vemos en la sección "Cartelera" y el segundo fue utilizado como background en la sección "Sobre Nosotros".
*En los botones de "Leer más" podemos ver aplicados los Modals Pop Up para mostrar una version extendida del texto visible en las secciones "Clases de teatro", "Círculo de debate" y "Sobre nosotros".
*Como fondo de la sección "Contacto y ubicación" podemos ver un patrón. Esto hecho gracias a la repetición de una pequeña imagen.
*Uso de elementos de , tales como la aparición del botón fijo para subir al início de la web a apartir de los 300px de scroll u ocultar el loader al haber cargado la página.
*Tambien contamos con el uso de variables de CSS (:root) para definir paleta de color y fuente a utilizar.
*Para hacer responsive se usaron múltiples @media queries para ajustar la disposición de elemntos, ocultarlos y adaptar el padding en dispositivos móviles.
*Uso de HTML semántico (<nav>, <main>, <section>, <footer>) y textos alternativos "alt" que describen las imagenes.
